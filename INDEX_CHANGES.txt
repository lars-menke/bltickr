AENDERUNGEN IN index.html

1) Vor dem grossen bestehenden <script> einfuegen:
<script type="module" src="./src/bridge.js"></script>

2) Diese Konstanten ersetzen:

ALT:
const API='https://api.openligadb.de';
const SEASON='2025';
const SERVER_URL='https://bltickr.fly.dev';
let curLeague=(['bl1','bl2'].includes(localStorage.getItem('bl_league'))?localStorage.getItem('bl_league'):'bl1');
let favorites = new Set(JSON.parse(localStorage.getItem('bl_favorites') || '[]'));

NEU:
const API = window.BLTICKR.APP_CONFIG.openLigaApi;
const SEASON = window.BLTICKR.APP_CONFIG.season;
const SERVER_URL = window.BLTICKR.APP_CONFIG.serverUrl;
let curLeague = window.BLTICKR.loadLeague();
let favorites = new Set(window.BLTICKR.loadFavorites());

3) Im Startpfad vor dem ersten Rendern Team-Icons fuer die aktive Liga laden:
await window.BLTICKR.loadTeamIconsForLeague({
  league: curLeague,
  season: SEASON,
  fetchJson: window.BLTICKR.fetchJson,
  apiBase: API,
});

4) Beim Ligawechsel ebenfalls nachladen:
await window.BLTICKR.loadTeamIconsForLeague({
  league: curLeague,
  season: SEASON,
  fetchJson: window.BLTICKR.fetchJson,
  apiBase: API,
});

5) In logo(team, right) NICHT mehr team.teamIconUrl direkt nehmen.

Stattdessen:
const iconUrl = window.BLTICKR.getResolvedTeamIcon({
  league: curLeague,
  season: SEASON,
  team,
});

const inner = iconUrl
  ? `<img src="${iconUrl}" alt="${a}" onerror="this.outerHTML='<span>${a}</span>'">`
  : `<span>${a}</span>`;

6) Falls noch direkter Re-Subscribe via fetch(`${SERVER_URL}/subscribe`, ...) im Code steht:
ersetzen durch
window.BLTICKR.subscribePush(existingSub.toJSON(), [...favorites])
  .then(d => { if (d?.key) localStorage.setItem('bl_push_key', d.key); })
  .catch(() => {});
